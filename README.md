# ECG data analysis

## [헬스케어] 성인 및 아동 심장질환 환자의 ECG 데이터 분석

### 초기 실행
```
git clone https://github.com/Julie35-jasmine/health-care-ecg-project.git
cd ecg_project

python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

### 작업 전
```
deactivate  # 가상환경 끄기(보통 할 필요 없음)
source .venv/Scripts/activate  # 가상환경 접속
git pull
```

### 작업 후
```
git add .
git commit -m '작업 내용'
git push
```
- 주의사항: git push하기 전 반드시 팀원들께 공유 후 push해 주세요!
